# FileServer
- 일종의 파일 스토리지, 업로드 된 파일을 따로 저장해놓는 서버
- 수업시간에는 프로젝트 파일 밖에 upload 폴더를 만들어서 업로드 된 파일 관리해 볼 것
```
클라이언트 - WAS - DB
          - FileServer
```

# 파일 업로드 후처리
- 파일이름이 중복되는 경우 업로드가 안 됨 > 겹치지 않을 랜덤 문자를 파일명에 붙이기
- 악성 코드가 업로드돼 컴파일되어 작동하는 경우 방지
